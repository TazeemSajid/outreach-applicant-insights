## 📅 Weekly Work Breakdown

### ✅ Week 1 — Data Quality Report
- Audited 3 raw datasets:
  - Applicant Data: 37,883 rows
  - Outreach Data: 37,887 rows
  - Campaign Data: 23 rows
- Issues identified & fixed:
  - Missing values & duplicate records
  - Invalid country entries (emails stored as country)
  - Inconsistent phone number formats
  - Inconsistent date formats
- Deliverables: 3 cleaned datasets + Data Dictionary

### ✅ Week 2 — EDA & Dashboard Report
- Full exploratory data analysis on cleaned datasets
- Key findings:
  - 65% of calls resulted in "Not Connected"
  - Only 2% of calls completed application
  - Peak outreach activity in March-April 2023
  - India (30.4%) and Bangladesh (25.2%) top countries
  - Significant caller performance variation identified
- Built initial dashboard wireframe

### ✅ Week 3 — Dashboard Development
- Connected master table to Google Looker Studio
- Built interactive outreach campaign dashboard with:
  - KPI scorecards (Total Outreach, Response Rate, Conversion Rate)
  - Record count over time (line chart)
  - Country distribution (donut chart)
  - Caller performance comparison (bar chart)
  - Campaign category breakdown
  - Interactive filters (Caller, Category, Outcome, Country)
  - Data table for detailed exploration

### ✅ Week 4 — Final Presentation & Deliverables
- Delivered insights and recommendations to stakeholders
- Presented findings on:
  - Caller performance variation
  - Country distribution concentration
  - Campaign effectiveness
  - Outreach growth trend

## 💡 Key Insights
- 65% of outreach calls result in no connection — major efficiency gap
- India, Bangladesh & Nepal account for 77% of all applicants
- Top callers (Prajwal, Poppy) achieve 50-60% positive response rate
- March-April are peak outreach months — clear seasonal pattern
- Only 2% escalation rate — complex cases may be underreported

---

## ✅ Recommendations
1. Reduce "Not Connected" rate through better call timing strategies
2. Diversify outreach beyond South Asia to grow applicant base
3. Share top caller best practices across entire calling team
4. Intensify campaigns during proven peak months (March-April)
5. Monitor conversion funnel to identify and fix drop-off points
